Book Tracker
Автор: Артём Романов
Язык: Python 3.x

Описание программы
Book Tracker — это простое приложение с графическим интерфейсом для учёта прочитанных книг.Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц,фильтровать список по жанру и количеству страниц. Все данные сохраняются в файл books.json.Проект использует систему контроля версий Git.

Инструкция по запуску
Установите Python 3.x (если не установлен).
Установите необходимые библиотеки:
bash
pip install -r requirements.txt  # (если потребуется)
(Для данного проекта требуется только стандартная библиотека Python)
Запустите программу:
bash
python book_tracker.py
Пользуйтесь интерфейсом для добавления и фильтрации книг.
Данные сохраняются автоматически при добавлении новой книги.
